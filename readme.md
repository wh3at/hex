# hex

A CLI automation workflow tool with ASCII art branding.

## Install

```bash
$ npm install --global hex
```

## CLI

```
$ hex --help

  Usage
    $ hex

  Options
    --help     Show help

  Commands
    /version   Show version information

  Examples
    $ hex
    Display hex logo and start the application

    $ hex /version
    Show version information
```

## Development

### Build

```bash
npm run build
```

### Development mode

```bash
npm run dev
```

### Test

```bash
npm test
```
